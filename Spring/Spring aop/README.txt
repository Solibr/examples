Код в состоянии связывания аспектов с помощью AspectJ. 
Для демонстрации работы аспектов рекомендуется запускать через mvnw package - на фазе тестирования отработает класс Runner.
Либо запустить собранный jar.

Для перехода на Spring AOP закомментировать в pox.xml aspectj-maven-plugin и раскомментировать зависимость spring-boot-starter-aop, а также аннотации @Component над аспектами в пакете com.example.aop.aspect
